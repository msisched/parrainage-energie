Obtenir un code de parrainage en 2 étapes

Comparez d’abord les offres, puis utilisez le code de parrainage du fournisseur choisi.

1

Comparez les fournisseurs

Il existe plusieurs fournisseurs d’électricité et de gaz sur le marché. Pour faire votre choix, vous pouvez utiliser le comparateur officiel du gouvernement.

Voir le comparateur officiel
2

Copiez le code de parrainage

Une fois le meilleur fournisseur choisi, souscrivez en copiant le code de parrainage indiqué dans le tableau ci-dessus.

Vérifiez toujours les conditions de l’offre avant de finaliser votre inscription.
