Comment utiliser un code de parrainage ?

Comparez d’abord les offres, puis utilisez le code de parrainage du fournisseur choisi.

1

Comparez les fournisseurs

Il existe plusieurs fournisseurs d’électricité et de gaz sur le marché. Pour choisir l’offre la plus adaptée à votre situation, vous pouvez utiliser le comparateur officiel du gouvernement.

Voir le comparateur officiel
2

Copiez le code de parrainage

Une fois votre fournisseur choisi, copiez le code de parrainage indiqué dans le tableau ci-dessous, puis utilisez-le au moment de la souscription.

Pensez à vérifier les conditions de l’offre avant de valider votre inscription.
